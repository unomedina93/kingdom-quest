🔒 Privacy for Parents

📷 Camera (Motion Control)

When enabled, the camera tracks your child's hand position only using MediaPipe — Google's on-device AI library. Video is never recorded, saved, or transmitted anywhere. Processing happens entirely on this device. The camera shows a small live preview in the corner so you can always see what's visible.

🎙️ Voice Commands

Voice commands use your browser's built-in speech recognition. In Google Chrome, audio snippets are sent to Google's speech servers for processing — this is a Chrome browser feature, not specific to this app. We recommend using voice commands only if you are comfortable with Chrome's privacy policy. Voice is off by default and must be manually enabled.

In Firefox or Safari, voice recognition runs fully on-device and nothing is sent externally.

💾 Data Storage

Progress (stars, completed levels, child's name) is saved only on this device using browser localStorage. Nothing is uploaded to any server. This app has no accounts, no analytics, and no internet connection requirement (except for loading fonts and MediaPipe on first use).

✅ Summary

  • No accounts or registration required
  • No ads, no tracking, no analytics
  • Camera: fully on-device (no cloud)
  • Voice: on-device in Firefox/Safari; Google servers in Chrome
  • All progress data stays on your device only
